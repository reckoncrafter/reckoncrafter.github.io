RUNEFOX🔗

Runefox is a toy Firefox extension that converts all onscreen characters into their closest equivalent character in runes. It's still a work in progress. The original program, which did this operation through a unix pipeline, was written in C++, and writing the an identical program in Javascript, even with outside code, proved to be difficult.
